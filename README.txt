This plugin collects parish information in one spot to be re-used throughout the website.

== Description ==
This plugin contains an area for Mass Times, Contact information, hours of operation and a Google Map.
Features a pop-up window that displays mass times and contact information.

== Installation ==

1. Upload `fim-parish-info.zip to the `/wp-content/plugins/` directory


== Development ==
This is a quick and dirty version just to get basic content up and running. A lot of refinement is still to come.
